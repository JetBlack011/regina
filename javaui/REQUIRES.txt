
Required libraries:

    regina.jar : main GUI
    regina-jni.jar : link to calculation engine using JNI
    regina-corba.jar : link to calculation engine using CORBA
        (at least one of regina-jni | regina-corba is required)
    btools.jar : miscellaneous utilities

Optional libraries:

    jpython.jar : allows JPython scripting
    jh.jar : JavaHelp internal help browser
    regina-docs.jar : documentation to view using internal help browser
