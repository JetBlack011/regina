Provides an interface to the calculation engine. This engine may be called in various ways, such as through JNI or across a network. Any class attempting to use the engine should use work through the general interfaces provided in this package, all of which are derived from ShareableObject. See the Java notes page in the engine documentation for further details.

Specific implementations of the engine can be found in the package normal.engine.implementation. @see normal.engine.ShareableObject
