Installing Cygwin

Cygwin provides a windows port of many of the tools that one generally finds on a GNU/Linux system. If you are using windows, Regina will require that you have Cygwin installed. You can download it from http://www.cygwin.com/.

You should then be able to run or compile Regina from within a cygwin shell the same way you would under linux.

When installing Cygwin you are offered a choice of several packages to install. Below is a bare minimum of packages that you must have installed.

  • Running Regina: ash, bash, cygwin, fileutils, gawk, libpopt0, libxml2, man, perl, sh-utils, textutils and which.
    Also recommended: gzip, tar, unzip and zip.
  • Compiling Regina: All of the above, plus binutils, diffutils, findutils, gcc2, grep, gzip, make, popt, sed, tar, unzip, zip and zlib.
    For compiling CVS snapshots: cvs, inetutils and openssh.
    Also recommended: less and vim.
Note that many of the other packages offered by Cygwin are also useful. Please mail me if you discover any additional packages that need to be added to the above lists.

GNU Multiple Precision Arithmetic Library

To compile the Regina calculation engine you will need the GNU Multiple Precision Arithmetic Library (libgmp). Note that Regina now requires shared libgmp libraries, unlike some previous versions of Regina which only required static libraries. There are two methods of obtaining the shared libgmp libraries:
  • Sources: You can download the libgmp sources from ftp://ftp.gnu.org/gnu/gmp/ and build the library yourself. Unpack the sources, move into the main source directory (the directory containing configure) and then issue the commands:
    • ./configure --prefix=/usr --enable-shared --disable-static
    • make
    • make install
  • Binaries: You can download prebuilt libgmp binaries. Visit the Regina download page and look in the support/cygwin module. You should find a prebuilt libgmp cygwin binary archive (gmp-version_cygwin-shared.tgz). Unpack this archive in your root cygwin directory using the following commands (issued from within a cygwin shell):
    • cd /
    • tar -zxf path/to/gmp-version_cygwin-shared.tgz
    where path/to is changed to reflect where you saved the archive and version is changed to reflect the name of the archive. You should now have the library /usr/bin/cyggmp-3.dll and the header /usr/include/gmp.h.
