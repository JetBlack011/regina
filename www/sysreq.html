You can either download prebuilt binaries or you can download the entire source tree to compile for yourself. System requirements are described below. If downloading a binary distribution, you need everything from the Binaries column. If downloading a source distribution, you need everything from both Binaries and Source columns.

Linux:

ComponentVersion Download From Binaries Source
GNU multiprecision arithmetic library>= 2 ftp://ftp.gnu.org/gnu/gmp/ or search your linux distribution Shared library libgmp.so Linkable library libgmp.a plus the corresponding header files
Java>= 1.3 is recommended but >= 1.2.2 will do http://www.blackdown.org/ Runtime environment (jre or j2re) Development kit (jdk or j2sdk); this will automatically include the runtime environment
BTools>= 1.1 or current CVS snapshot http://sourceforge.net/projects/btools/ Regina only needs access to the single file btools.jar.
Jython (optional - used to run scripts or otherwise obtain low-level access to the computation engine)>= 2.1-alpha1 http://www.jython.org/ Regina only needs access to the single file jython.jar.
JavaHelp (optional - used to view documentation through the embedded help system)>= 1.1 http://www.javasoft.com/products/javahelp/ Regina only needs access to the single file jh.jar.
omniORB (optional - used to run the calculation engine as a CORBA server)>= 3 http://www.uk.research.att.com/omniORB/omniORB.html Shared libraries libomniORB*.so et al plus a name server if necessary Linkable libraries libomniORB*.a et al plus the corresponding header files
Development tools Search your linux distribution g++, a GNU-compatible make, bash, cat, cp, echo, grep, pwd, rm, sed, tar, touch and almost certainly other bits and pieces; please mail me if you discover any version requirements or other utilities that need to be added to this list.
Doxygen (optional - used to make HTML documentation for the computation engine)>= 1.2.2 http://www.stack.nl/~dimitri/doxygen/ or search your linux distribution A binary doxygen package plus graphviz (see below)
Graphviz (optional - required if doxygen is used; see above)>= 1.5 http://www.research.att.com/sw/tools/graphviz/ or search your linux distribution A binary graphviz package

Windows (at least Win95):

I still haven't actually tried building under windows. In theory the sources should compile fine on windows assuming tools similar to those listed above are available on the system (the Cygwin development tools are a great place to start).
