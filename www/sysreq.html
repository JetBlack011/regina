If you just wish to run Regina, you can download prebuilt binaries for either Linux or Windows. If you wish to view or play with the sources or if you're just that way inclined, you can download the entire source tree to compile for yourself.

System requirements are described below. If downloading a prebuilt binary distribution, you need everything from the Binaries column. If downloading a source distribution, you need everything from both Binaries and Source columns.

Linux:

Many of the tools in the list below may come with your linux distribution; it may be worth searching there before visiting each individual website and installing each individual piece of software. Certainly debian (unstable) offers all of the tools in this list except for JavaHelp.

Windows (at least Win95):

Although Regina is primarily written and tested under linux, it has been made portable enough to build and run under windows. The current status of windows support is as follows:
  • Calculation engine (JNI): builds, runs
  • Calculation engine (CORBA): build not tested
  • Java user interface (JNI): builds, runs
  • Java user interface (CORBA): builds, run not tested
