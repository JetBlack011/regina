If you just wish to run Regina, you can download prebuilt binaries for either Linux or Windows. If you wish to view or play with the sources or if you're just that way inclined, you can download the entire source tree to compile for yourself.

System requirements are described below. If downloading a prebuilt binary distribution, you need everything from the Binaries column. If downloading a source distribution, you need everything from both Binaries and Source columns.

Linux:

Many of the tools in the list below may come with your linux distribution; it may be worth searching there before visiting each individual website and installing each individual piece of software. Certainly debian (unstable branch) offers all of the tools in this list except for JavaHelp.

If you wish to generate the documentation (such as the calculation engine docs or the HTML reference manual), see the separate documentation section below.

Windows (at least Win95):

Although Regina is primarily written and tested under linux, it has been made portable enough to build and run under windows. The current status of windows support is as follows:
  • Calculation engine (JNI): builds, runs
  • Java user interface (JNI): builds, runs

If you wish to generate the documentation (such as the calculation engine docs or the HTML reference manual), see the separate documentation section below.

Generating Documentation:

If you'd rather regenerate the documentation from the sources instead of downloading a prebuilt documentation bundle, you will need the following tools.

Note that you might not need all of the tools in this list. For instance, if you only wish to regenerate the calculation engine docs (perhaps you've modified the C++ sources), you will only need Doxygen and Graphviz (but then you should be careful to invoke make docsengine instead of make docs to avoid rebuilding the other components of the documentation).

ComponentVersion Reason Download From
Doxygen >= 1.2.2 Calculation engine docs http://www.stack.nl/~dimitri/doxygen/
Graphviz>= 1.5 Calculation engine docs http://www.research.att.com/sw/tools/graphviz/
JavaDoc>= 1.3 Java UI class docs (should come bundled with your Java compiler)
DocBook SGML DTD [+]4.1 HTML reference manual http://www.oasis-open.org/docbook/sgml/4.1/
DocBook DSSSL Stylesheets [+]>= 1.73 HTML reference manual http://sourceforge.net/projects/docbook-dsssl
ISO Entity Sets [+] HTML reference manual http://www.oasis-open.org/cover/ISOEnts.zip
Jade [+]>= 1.2.1 HTML reference manual http://www.jclark.com/jade/
Perl [++]>= 5 Man pages http://www.perl.com/pub/a/language/info/software.html
SP (SGML parser) [++] Man pages http://www.jclark.com/sp/
SGMLSpm (Perl modules) [++]>= 1.03ii Man pages ftp://ftp.kde.org/pub/kde/devel/docbook/SOURCES (file SGMLSpm-version.tar.gz)

[+] HTML reference manual:

The DocBook-related tools are not at all easy to set up. If you are using Linux, it is strongly recommended that you search your Linux distribution for these tools instead of downloading them from the sites provided. Hopefully they will come preconfigured.

For Debian users, the Debian packages you will need are docbook, docbook-dsssl, sgml-data and jade.

If you are using Windows, you should visit the DocBook support page for instructions on configuring these tools correctly.

[++] Man pages:

Building man pages requires DocBook Utils; the other tools marked [++] are merely to support DocBook Utils.

Building man pages is currently not supported under Windows. Again, if you are using Linux, it is strongly recommended that you search your Linux distribution for these tools instead of downloading them from the given sites.

Debian users should simply install Debian package docbook-utils and its dependencies.
