SnapPea Kernel (Unused Files)
-----------------------------

This directory contains portions of the SnapPea kernel, version 3.0d3.

The files in this directory are not built into Regina, and are not
distributed as part of Regina's source downloads.  They are kept here
in the online source repository for reference and possible future use.

For SnapPea files that are built into Regina, see the parent directory.
