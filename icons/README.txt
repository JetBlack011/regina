Regina Icons Directory
----------------------

This directory contains a set of icons that you may use on your
desktop/panel/etc to run Regina.

  reginalarge.* : 150x150 high-colour image
  regina.* : 32x32 high-colour image
  regina-lo.* : 32x32 low-colour image
  reginatiny.* : 16x16 high-colour image
  reginatiny-lo.* : 16x16 low-colour image
